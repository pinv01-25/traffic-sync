Fuzzy Inference System Core
===========================

This module defines the fuzzy inference system including:

- Fuzzy variables: vehicles per minute, speed, density, congestion.
- Membership functions (trapezoidal, gaussian, triangular).
- 27 fuzzy rules covering all input combinations.
- Control system and simulator initialization.
