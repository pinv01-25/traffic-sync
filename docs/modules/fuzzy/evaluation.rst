Traffic Congestion Evaluation Engine
====================================

This module provides the `evaluate_congestion()` function and test cases used to validate fuzzy logic outputs.

It returns:

- The crisp congestion level.
- The linguistic category with the highest membership.
- A breakdown of fuzzy memberships for interpretability.
