Fuzzy Logic Congestion Classifier
=================================

This module implements a Mamdani-type fuzzy logic system to classify urban traffic congestion into levels: **none**, **mild**, and **severe**. The system uses linguistic rules and membership functions based on real-world traffic indicators.
