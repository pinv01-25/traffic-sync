Hierarchical Clustering Visualizations
======================================

This module handles the visual rendering of hierarchical clustering results, including:

- A dendrogram plot built with Plotly and SciPy linkage data.
- Custom styling, axis labeling, and threshold annotations.
- Subtitle metadata showing clustering method and dissimilarity settings.
- Local caching of the generated figure for performance.
